This application is an angularjs single page application built on top of MVC

Please note that the data for this application is stored in a json file called Employee.json

Steps to run this  app

1. To run this program, create a folder C:\Assessment and copy the Employee.Json file to that directory. 

2. Go to the root folder of EmployeeApp_API project and run "npm install --save" command to Download all the angular packages

Note also that all javascript references in the bundleConfig.cs file.

Run the application and navigate to http://localhost:64808/#/
